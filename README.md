# Pokedex
First generation pokedex using ascii prints and built-in webbrowser module.
# Run The Source Code
#### Download Python
https://www.python.org/downloads/
#### Verify Version Once Installed
py --version
#### From Your Download Directory Run The Source Code
py pokedex.py
